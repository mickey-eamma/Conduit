# Conduit

Monorepo for the Conduit application.

## Layout

- [`frontend/`](frontend) — React + TypeScript + Vite web app. See [frontend/README.md](frontend/README.md).
- [`backend/`](backend) — backend service (not yet implemented).
- [`infrastructure/`](infrastructure) — deployment/infrastructure code (not yet implemented).

## Getting started

From the repo root:

- `setup.bat` — installs Node.js (if needed) and the frontend's dependencies.
- `run.bat` — starts the frontend dev server and opens it in your browser.

These currently set up and run the frontend only; backend/infrastructure tooling will be added here as those parts of the monorepo are built out.
